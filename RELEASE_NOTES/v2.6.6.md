## 更新内容

### 🚀 重大更新
- 新增“正文标点标准化”能力：中文正文中的常见英文半角标点会自动规范为中文标点，并统一作用于预览、复制和同步结果。
- 右侧排版 setting panel 新增“正文标点标准化”开关，支持即时切换并实时预览结果。

### 🐛 问题修复
- 修复中文省略号 `...` / `......` 被错误拆分替换的问题。
- 修复技术写作中的括号记号误改问题，例如 `公式(x+y)`、`矩阵(A,B)`、`变量(x)` 等表达式现在会保持原样。
- 补齐 triplet renderer 的防御性错误分支测试，覆盖无 DOM、缺少 converter、legacy render 缺少 app 等场景。

### ✨ 功能优化
- 为 URL、邮箱、路径、命令行参数、环境变量、时间日期等技术内容增加保护规则，降低误改风险。
- 设置面板中的“正文标点标准化”位置已调整到 “Mac 风格代码块” 上方，并去掉“默认开启”的提示文案。
- 更新 README 中的设置面板截图与“中文标点标准化规则”说明。
- 发布流程已调整为 Obsidian 社区插件兼容模式，后续将使用与 `manifest.json.version` 一致的无前缀版本 tag。

## Updates

### 🚀 Major Updates
- Added the experimental AI layout workflow to the converter panel, including provider management, built-in layout families, schema checks, cached results, and debug prompt export.
- Added WeChat-safe Mermaid export: diagrams remain visible in Obsidian preview, then rasterize to PNG during copy and draft sync.
- Added account-level draft defaults for supported WeChat fields such as source URL, comment settings, and fans-only comments.

### ✨ Improvements
- AI layout results now keep the latest cache per layout family and can be applied directly when they still match the current article.
- AI layout colors can be selected before generation or changed after generation without rerunning the full layout plan.
- Regeneration failures now preserve the previous successful AI layout when available.
- Quote and callout styling now includes a neutral gray mode and semantic callout accents for common types.
- The AI layout panel has clearer status text, loading copy, cache labels, debug copy actions, and spacing.

### 🐛 Fixes
- Fixed Mermaid copy and draft sync so exported diagrams are treated as images instead of fragile SVG payloads.
- Fixed copy feedback so the toolbar uses a lightweight loading state without duplicate progress messages.
- Fixed AI layout panel flicker when switching between notes with and without cached layout results.
- Fixed duplicate failure copy in the AI layout status card.
- Improved WeChat compatibility for code block controls, Mermaid output, math alignment, and unknown callout rendering.

### 📚 Documentation
- Updated English and Chinese README files for AI layout, Mermaid export, account-level draft defaults, and the 2.7.0 release scope.

## 更新内容

### 🐛 问题修复
- **修复 AI Provider 连接偶发失败**：增强 Obsidian `requestUrl` 适配器，兼容部分第三方 OpenAI 中转服务返回的非标准 JSON 文本，减少 `Unexpected non-whitespace character after JSON` 这类解析失败。
- **增加纯文本请求回退**：当 `requestUrl` 在内部 JSON 解析阶段失败时，会自动回退到 Obsidian `request()` 获取原始文本，再由插件进行更稳健的解析。

### ✨ 功能优化
- **提高 Anthropic 输出上限**：AI 编排的 Anthropic `max_tokens` 从 4096 提升到 8192，降低复杂文章排版 JSON 被截断的概率。
- **优化 Anthropic 截断提示**：当 Anthropic 返回 `stop_reason: "max_tokens"` 时，会显示明确的输出上限提示，而不是落到模糊的 JSON 解析错误。

### 🧪 测试
- 新增 AI Provider 响应解析、`requestUrl` 解析失败回退，以及 Anthropic 输出上限的回归测试。
